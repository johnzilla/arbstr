| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-23-07 | I (Information Disclosure) | VAULT_INTERNAL_TOKEN | mitigate | Token passed via environment variable, not embedded in config or image. Docker secrets would be better but is overkill for v2.0 single-node deployment. |
