## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-21-03 | Spoofing | agent_token in reserve | mitigate | Vault validates agent_token server-side; proxy only forwards it. Tests verify 401 when token missing. |
| T-21-04 | Information Disclosure | vault error messages | mitigate | Vault error details wrapped in generic OpenAI error format, not forwarded raw. Tests verify error format. |
| T-21-05 | Denial of Service | vault rate limiting | mitigate | 429 from vault propagated to client. Backpressure check prevents request accumulation. |
